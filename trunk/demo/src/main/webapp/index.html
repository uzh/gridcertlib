GridCertLib demo servlets

The following links will perform SLCS certificate retrieval and VOMS proxy creation, through servlets implemented on top of the GridCertLib library.

Note: For the servlets to work properly, access to this URL space must be Shibboleth-protected and valid Shibboleth credentials must be available to the servlet container.

  • Get a new SLCS certificate, print paths to the PEM-encoded certificate and private key files into browser window.
  • Get a new proxy for the smscg VO; print path to the PEM-encoded proxy file into browser window. (This is just to show what a VomsProxyInit request URL looks like; you need to edit the URL and supply correct values for all the fields for it to work.)
