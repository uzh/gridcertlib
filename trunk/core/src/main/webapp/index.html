GridCertLib support servlets

The following links will perform Shibboleth SAML2 credential renewal, through servlets implemented on top of the GridCertLib library.

Note: For the servlets to work properly, access to this URL space must be Shibboleth-protected and valid Shibboleth credentials must be available to the servlet container.

  • Get a fresh SAML2 assertion (performs a Shibboleth logout and then login again)
